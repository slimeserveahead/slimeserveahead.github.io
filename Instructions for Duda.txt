To host your HTML, CSS, and JavaScript files on Duda, you can follow these steps:

Create an account on Duda if you don't already have one.

Click on the "New Website" button and select "Start from Scratch".

Choose a template that you like, or select "Blank Site" if you want to start from scratch.

Once you have selected a template or created a blank site, you will be taken to the Duda editor.

Click on "Pages" in the left-hand menu and then click on "New Page" to create a new page.

Give your new page a name and click "Create".

Now you can start adding your HTML, CSS, and JavaScript to the page.

To add HTML, click on the "HTML" element in the left-hand menu and drag it onto the page. You can then paste your HTML code into the editor.

To add CSS, click on the "Design" element in the left-hand menu and then click on "Custom CSS". You can then paste your CSS code into the editor.

To add JavaScript, click on the "Code" element in the left-hand menu and then click on "Custom Code". You can then paste your JavaScript code into the editor.

Once you have added all of your HTML, CSS, and JavaScript, click on the "Publish" button to make your website live.
